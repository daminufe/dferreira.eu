“Progressive Web Apps - an introduction to a game changer”

Progressive Web Apps are experiences that combine the best of the web and the best of apps. They are useful to users from the very first visit in a browser tab, no install required. As the user progressively builds a relationship with the App over time, it becomes more and more powerful. It loads quickly, even on flaky networks, sends relevant push notifications, has an icon on the home screen and loads as a top-level, full screen experience.

“Quickly prototype Full-Stack Web apps with JS only”

On a world where creating startups is getting high popularity, web developers find a certain difficulty to quick prototype a web app from the database, server and fronted.
In this live coding session will be shown how to do it quickly using the full-stack Javascript. The tech stack for this talk is Node.JS, Restify, Angular and MongoDB

“Ionize your web app - build hybrid apps with Typescript”

Free and open source, Ionic 2 offers a library of mobile-optimized HTML, CSS and Typescript components and tools for building highly interactive native and progressive web apps.
Built with Sass, optimised for Angular 2, Ionic 2 is modeled off of standard native mobile development SDKs, bringing the UI standards of native apps together with the full power and flexibility of the open web.
Ionic runs inside Cordova or Phonegap to deploy natively, or as a Progressive Web App.
Develop once, deploy everywhere.
